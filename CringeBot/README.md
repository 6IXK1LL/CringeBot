# 🤦‍♀️ Kreange

> What is it?

> Kreange - bot for moderation in discord chats.

---

> How to activate a bot?

### <b>Linux</b>:
```bash
venv/bin/activate
```
```bash
python3 main.py
```

### <b>Windows</b>:
```bash
pip install colorama discord.py discord-components 
```

```bash
py main.py
```
